# 笔记本 & 日历应用

一个轻量级的本地优先任务管理应用，具有笔记本和日历视图。

## 功能特性

- 创建、编辑和删除任务，支持富文本样式
- 每个任务最多上传 3 张图片
- 标记任务为完成/未完成
- 筛选任务（全部/进行中/已完成）
- 按标题、内容或标签搜索任务
- 月历视图
- 任务优先级（高/中/低）
- 任务模板快速创建
- 本地文件存储（无需数据库）
- 导入/导出功能
- 精美的现代化界面，流畅动画

## 快速开始

### 🚀 简单方式（Windows）

**只需双击 `start.bat`** - 它会自动启动前后端服务器！

应用将在以下地址打开：
- 前端：http://localhost:3000
- 后端：http://localhost:3001

### 📋 手动设置

如果您更喜欢手动运行服务器：

#### 后端

```bash
cd backend
npm install
npm start
```

服务器运行在 http://localhost:3001

#### 前端

```bash
cd frontend
npm install
npm run dev
```

应用运行在 http://localhost:3000

## 使用方法

1. 启动应用（双击 `start.bat` 或手动运行服务器）
2. 在浏览器中打开 http://localhost:3000
3. 点击"新建任务"创建任务
4. 在笔记本和日历视图之间切换
5. 使用筛选器显示全部/进行中/已完成任务
6. 使用搜索栏搜索任务
7. 将常用任务保存为模板

## 功能详解

### 任务管理
- **创建**：添加任务，包含标题、描述、图片、截止日期、标签和优先级
- **编辑**：修改任何任务属性，添加/删除图片
- **删除**：删除任务（需确认）
- **切换完成状态**：标记任务为完成/未完成
- **优先级**：设置高/中/低优先级，带颜色标识

### 视图
- **笔记本**：网格视图显示所有任务，支持筛选和搜索
- **日历**：月历显示按截止日期排列的任务

### 组织管理
- **搜索**：按标题、内容或标签查找任务
- **筛选**：显示全部、进行中或已完成任务
- **标签**：使用逗号分隔的标签组织任务
- **优先级**：可视化优先级指示器

### 模板
- 保存常用的任务格式
- 从保存的模板快速创建
- 本地存储在浏览器中

### 自定义
- **文字颜色**：从 10 种预设颜色中选择
- **字体粗细**：正常或粗体
- **图片**：每个任务最多 3 张图片

## API 接口

- `GET /api/tasks` - 获取所有任务
- `POST /api/tasks` - 创建任务
- `PUT /api/tasks/:id` - 更新任务
- `PATCH /api/tasks/:id/done` - 切换完成状态
- `DELETE /api/tasks/:id` - 删除任务
- `GET /api/export` - 导出任务为 JSON
- `POST /api/import` - 从 JSON 导入任务

## 数据存储

所有数据都本地存储在 `/data` 文件夹中：
- 任务元数据：`/data/tasks.json`
- 图片：`/data/images/`

无需外部数据库或云存储。

## 使用技巧

- 任务自动排序：未完成 → 优先级 → 截止日期
- 图片限制为每张 5MB
- 模板保存在浏览器的 localStorage 中
- 使用标签更好地组织任务
- 日历视图显示任务的截止日期
- 搜索功能覆盖标题、内容和标签

## 语言支持

应用支持中英文双语：
- 点击顶部导航栏的语言按钮切换
- 默认语言：英文（Eng）
- 可切换至：中文（中文）
- 语言选择会自动保存

## 离线使用

此应用完全支持离线使用：
- 所有数据存储在本地
- 无需互联网连接
- 图片保存在本地文件系统
- 模板保存在浏览器中
